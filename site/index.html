    PE (or Poly Electrolyte, if you want to define the acronym) is a powerful, coarse-grained simulation software first designed for modelling of complex polyelectrolytic systems.

    üí° What is Coarse-Grained Simulation?

    Coarse-grained simulation simplifies complex systems by grouping multiple atoms or elements into single, larger "particles." This approach significantly reduces computational complexity, allowing us to:

    • Run simulations much faster than all-atom models.
    • Model systems at larger scales.
    • Focus on macroscopic behaviors and properties.

    üöÄ Where to Start

    We recommend new users start with the First Run Example. This simple, guided tutorial confirms your setup is correct and provides a working baseline before you dive into complex configurations.
